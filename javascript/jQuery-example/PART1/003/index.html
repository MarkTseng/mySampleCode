何謂Foundation

Founction是由ZURB公司所開發、發佈的一種前端框架，可有效率的以"Mobile First"製作"Resposive" Web頁面。
其能使用類似於自然語言的方式撰寫CSS類別等優點，能讓WEB開發非常有效率。

前端框架

所謂前端框架，簡單來說就是CSS的預設集合，將一些表格系統、表單元素等基本Layout以系統化的方式提供類別使用。

優點

藉由利用前端框架，可利用固定的規定進行配置與設計，開發者就可更專注於更重要的開發工作。

Resposive

Resposive設計是可讓網頁內容以適合設備的方式做顯示的一種設計模式。使用如Media Query 等CSS3的機制，讓內容可在設備上適當的顯示。

Mobile First

網際網路已經進入大量使用移動設備存取的時代，「首先製作移動設備用的網頁」就是Mobile Fisrt的一種概念。

HTML5/CSS3

支援這些做法的基礎就是HTML5與CSS3的Web標準。依據這些Web標準所發展出來的技術，不論是線上或離線的，都可以在不同的地方發現。
